# AiYo v0.3.8

将 Supertester 插件市场源迁移至自建 Git 仓库。

## Changes

- **Supertester 市场源更新**: 将内置的 supertester 市场从 `supertester-ai/supertester` (GitHub) 迁移到 `https://git.vemic.com/MIC/mictest/supertester.git` (自建 Git)

## Notes

- 新仓库需包含 `.claude-plugin/marketplace.json` 文件，且 `name` 字段需为 `"supertester"` 以保持插件 ID `supertester@supertester` 可正常解析
- 如果仓库为私有，需确保 sidecar 运行环境具备相应的 git 认证凭据
- 克隆失败不会导致应用启动崩溃，仅输出错误日志
