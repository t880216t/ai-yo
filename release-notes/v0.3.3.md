# AiYo v0.3.3

本次版本主要完成了项目品牌升级：从 "Claude Code Haha" 更名为 **AiYo**，同步更新了应用图标、产品名称和界面文案。

## Highlights

- **品牌升级**：产品名称从 "Claude Code Haha" 更名为 "AiYo"，桌面端应用图标全面更新。
- **界面文案调整**：侧边栏、标题栏和设置页面中的品牌相关文案同步更新。
- **多平台图标适配**：macOS (icns)、Windows (ico)、iOS 和 Android 各尺寸图标均已更新为新品牌图标。

## Notes

- 本次为品牌升级版本，功能层面与 v0.3.2 保持一致。
- GitHub Release 正文以 `release-notes/v0.3.3.md` 作为来源。

## macOS

首次打开如果提示"已损坏"或"无法验证开发者"，请执行：

```bash
xattr -cr /Applications/AiYo.app
```
