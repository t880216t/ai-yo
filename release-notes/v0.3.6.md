# AiYo v0.3.6

扩展内置插件能力与技能管理体验优化。

## Changes

- **内置插件扩展**: 新增 12 个内置插件（office-documents、legal-cn、superpowers、context7、github、frontend-design、supabase、code-review、commit-commands、skill-creator、playwright、typescript-lsp/jdtls-lsp/pyright-lsp），启动时自动安装
- **插件市场预注册**: 内置注册 claude-office-skills、claude-plugins-official 等市场源
- **技能管理提示优化**: 设置页"已安装技能"下方文案动态显示实际 config 目录路径，替代硬编码的 `~/.claude/skills/`

## Notes

- 内置插件在 sidecar 启动前通过 CLI 模式安装，确保服务启动后插件立即可用
- 技能路径文案结合 `activeConfigDir`（优先环境变量 `CLAUDE_CONFIG_DIR`，回退系统 `app_config_dir`）
